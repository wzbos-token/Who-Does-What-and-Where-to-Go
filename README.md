Stepwise‑ME 是一个专为多实体 3D 轨迹生成设计的大规模数据集，旨在解决复杂交互场景中的身份‑动作绑定与空间碰撞问题。数据集包含超过 31,000 个视频片段，总计约 800 万帧，覆盖 12 个同步视角，每段平均时长 3.3 秒（下采样至 8 FPS）。其核心特色在于逐步骤（stepwise）的细粒度文本标注，明确分解了 45 种空间平移组合与 26 种速度修饰，使得运动指令不再局限于静态描述，而是反映动态时序过程。与现有场景中心或单实体数据集（如 RealEstate10k、CCD、E.T.）不同，Stepwise‑ME 聚焦于多实体自由运动，是首个严格支持时空对齐的多实体轨迹生成基准，为具身世界模型和可控视频生成提供了关键的数据支撑。
