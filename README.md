# Who-Does-What-and-Where-to-Go


## Dataset  Stepwise-ME

**下载链接**：https://pan.baidu.com/s/1VzAWJMEodwN7Wxt-w7rAmQ?pwd=mggn  
**提取码**：`mggn`

*若链接失效，请及时联系维护者更新。

- **用途**：多实体 3D 轨迹生成（支持交互式视频生成与具身世界模型）
- **规模**：31,000+ 视频片段，约 800 万帧，12 个同步视角
- **时长**：平均每段 3.3 秒（下采样至 8 FPS）
- **标注**：**逐步骤（stepwise）** 细粒度文本描述，包含 45 种空间平移组合 + 26 种速度修饰
- **特点**：多实体、自由运动、面向运动指令（非静态描述），首个严格支持细粒度时空对齐的多实体轨迹数据集
- **对比优势**：相比现有场景中心或单实体数据集（如 RealEstate10k、CCD、E.T.），本数据集明确支持多实体交互与逐阶段动作分解
